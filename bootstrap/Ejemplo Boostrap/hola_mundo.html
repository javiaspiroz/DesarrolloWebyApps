Galería de Imagenes 1


Se usa la clase img-fluid para hacer que las imagenes sean responsive

Para usar el sistema de rejillas o grid de Bootstrap, debemos poner el contenido dentro de una class container

Dentro del container creo las filas, dentro de cada fila las diferentes columnas. Recordad que son 12 columnas. En el ejemplo siguiente colocamos 3 imagenes, cada imagen ocupa 4 columnas (4+4+4 = 12)

Las columnas son nombradas dependiendo del tamaño de las pantallas. En el ejemplo a continuación usamos col-md para decir que se comportará así si las pantallas son mayo o igual que 768 px. Para más información sobre Grid visita Grid en Bootstrap
